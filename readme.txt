== Description ==

Cassandra is a responsive and clean WordPress Theme. It is easily customizable and comes with many great features. It is designed for your blog, portfolio or news magazine. It has a unique, modern, light and clean design which will make your website look awesome. It is completely built on Customizer which allows you to customize most of the theme settings easily with live previews.
